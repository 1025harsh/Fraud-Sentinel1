# Fraud Sentinel

## Overview
Fraud Sentinel is a real-time credit card fraud detection and monitoring system designed to identify suspicious transactions and alert administrators.

## Features
- User Authentication
- Fraud Risk Scoring
- Transaction Monitoring
- Fraud Case Management
- Email Alerts
- Dashboard Analytics
- Notification System

## Technologies Used
- React
- Node.js
- Express.js
- PostgreSQL
- Replit

## How It Works
1. User logs into the system.
2. Transactions are monitored in real time.
3. Risk scores are calculated.
4. Suspicious transactions are flagged.
5. Alerts are sent to administrators.
6. Fraud cases can be reviewed and managed.

## Future Enhancements
